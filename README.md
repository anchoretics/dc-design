# dc-design

## description
    项目为自定义组件示例：
    src/components/dc-table.vue  // 表格组件
    表格功能包含基本样式、带边框、带条纹状、单元格省略、排序、行选择、分页、自定义列的渲染
    支持PC、移动端响应式布局

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
